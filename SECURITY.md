# Security Policy

## Supported Versions

The following versions of this project are currently being supported with security updates:

| Version | Supported          |
| ------- | ------------------ |
| 0.1.0   | :white_check_mark: |

## Reporting a Vulnerability

If you discover a security vulnerability within this project, please **do not** report it via a public issue. Instead, please use the GitHub "Report a vulnerability" button on the Security tab of this repository.

Alternatively, if private reporting via GitHub is not possible, please contact the maintainers directly.

We take all security reports seriously and will work to address them as quickly as possible.
